html을 꾸며주는 언어

기초문법

css는 태그를 선택해서 스타일을 입력하는 언어

태그를 선택하는 부분과 스타일이 입력되는 부분으로 나뉜다

선택자{스타일속성:스타일속성값;스타일속성:스타일속성값}

스타일 입력 방식

  • 인라인 스타일 방식 - 태그에 직접 스타일속성을 사용
  • 내부 스타일시트 방식 - style 태그를 만들어서 스타일을 입력
  • 외부 스타일시트 방식 - style 파일을 만들어서 연결

인라인 스타일 방식의 단점

소스코드의 가독성을 해친다

태그를 하나하나 선택하여 스타일을 입력하는 방식이기 떄문에, 일괄 수정이 불가능하다.

내부 스타일 시트 방식의 단점

스타일태그가 html파일 안에 있기 때문에 여러 페이지를 동시에 스타일 수정을 할 수 없다.
