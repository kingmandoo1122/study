float

선택한 태그를 공중에 붕~ 띄워서 왼쪽 혹은 오른쪽에 배치시키는 방식

공중에 붕~ 띄워진 태그들은 공간을 차지하지 않고 부모태그를 기준으로 했을 때, 집나간 자식태그가 된다. -> 부모태그가 자식태그를 포함할 만큼 높이를 자동으로 만들 수 없다. -> 부모태그의 다음형제태그가 있을 때 레이아웃이 틀어질 수 있다.

clear

float를 해제할 때 사용하는 속성

clear 속성이 적용된 태그는 앞서 만들어진 형제태그 중에서 float 되어있는 태그 중 가장 아래까지 내려와있는 태그의 바로밑에 배치된다.
