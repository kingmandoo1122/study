비디오태그에 사용할 수 있는 속성들

속성들은 석택적으로 사용해야 한다. 예를들어 동영상을 홈페이지의 배경이미지 처럼 사용하고 싶다면, controls 속성은 빼주는게 낫다. 왜냐하면 controls 속성이 명시되어 있으면 영상부분을 클릭했을 때 일시정지가 되기 때문이다.

비디오태그도 이미지태그나 글자처럼 컨텐츠일뿐 맥락이 있지 않다.

  • controls -> 비디오제어박스
  • autoplay -> 자동재생
  • muted -> 음소거
  • loop -> 반복재생
  • poster -> 섬네일지정

source 태그를 사용하여 여러가지 타입의 영상
