transition

중간수치로 표현가능한 css속성들에 대해서 처음과 끝 사이의 중간 스타일을 자동으로 애니메이션처럼 만들어주는 속성

변하기 전 스타일에 적용

개별 속성

  • transition-property - 전환속성을 지정할 css 속성을 선택(all - 모든 css 선택)
  • transition-duration - 전환스타일을 몇 초동안 보여줄 것인지 설정(1s=1초=1000ms, 0.5s=0.5초=500ms)
  • transition-timing-function - 전환 애니메이션의 속도감을 설정 ( linear 일정한 속도) / (ease, ease-in, ease-in-out)
  • transition-delay - 전환 스타일을 몇초 후에 실행할 것인지 설정(s,ms)
