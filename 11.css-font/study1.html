font

  • font-size -> 글씨의 크기를 지정하는 속성

    브라우저(html태그)의 글씨크기 기본값은 16픽셀이다.

    글씨의 가장 높은 부분에서 낮은 부분까지의 길이를 기준으로 한다.(글씨체에 따라 다르게 적용될 수 있다.)

    1em -> 부모태그에 적용된 글씨크기의 1배만큼

    1rem -> html태그에 적용된 글씨크기의 1배만큼

  • line-height -> 행간(글씨를 감싸는 보이지않는 라인의 높이)

    행간은 기본적으로 글씨크기의 120%만큼 설정되어 있다.행간은 기본적으로 글씨크기의 120%만큼 설정되어 있다.

    120%는 소수로 1.2라고 표현해도 된다. 글씨크기가 50픽셀만큼 설정되어 있을때 line-height가 3만큼 지정되어있다면 계산된 행간은 150픽셀이 된다. 글씨크기를 제외한 100픽셀을 위아래로 똑같이 나누어 여백을 만들게 된다.

  • font-weight

    글씨의 굵기 설정

    사용하는 글씨체마다 호환되는 글씨의 굵기가 다르기 때문에 이를 잘 활용하여 사용해야 한다.

    • bold - 두껍게
    • normal - 기본값
    • lighter - 얇게
    • 100~900 까지 100단위의 숫자를 사용해서 지정가능 - 숫자가 커질수록 굵어짐(기본값400)
