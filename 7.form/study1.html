양식태그

홈페이지를 사용하는 사용자가 내용을 입력할 수 있도록 양식을 제공할 때 사용하는 태그

label태그는 입력박스에 대한 이름표태그

label태그와 input태그를 서로 연결하기 위해 input태그의 id속성과 label태그의 for속성 값을 똑같은 값으로 적용

label태그가 눈에 보이지 않아도 css에서 안보이게 처리해야 한다. 지우면 안된다.


input태그 중 type 속성이 text로 적용되어 있으면 짧은 한 줄의 텍스트를 입력받을 용도로 사용한다.

input 태그의 type 속성에 적용되는 속성 값

  • text - 짧은 한 줄의 텍스트 입력
  • password - 비밀번호를 입력
  • radio - 필수적으로 선택해야할 항목(해제가 안됨)
  • checkbox - 자유롭게 선택할 항목(해제가 됨)
  • number - 숫자만 입력받는 용도로 사용(모바일 환경에서는 숫자패드가 올라옴)

사용되는 속성

  • value - 입력박스에 들어가는 실제정보/버튼형태에서는 버튼내용
  • disabled - 선택한 입력박스를 비활성
  • required - 선택한 입력박스 필수정보를 입력받도록
  • selected - option태그가 선택되도록
  • checked - radio/checkbox 버튼이 체크가 되도록
